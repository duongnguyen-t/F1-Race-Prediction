# F1 Race Prediction

End-to-end machine learning system to predict Formula 1 race outcomes and optimize pit stop strategies using real-world racing data.

## Overview
This project builds a predictive analytics pipeline that integrates race results, weather conditions, and pit stop data to model race outcomes and recommend strategies.

## Key Features
- API-based data ingestion from OpenF1
- Feature engineering on driver, team, and race conditions
- Machine learning models (Random Forest, Gradient Boosting)
- Pit stop strategy optimization system
- Interactive visualizations with Plotly

## Tech Stack
- Python (pandas, NumPy, scikit-learn)
- Data pipelines & API integration
- Plotly, matplotlib for visualization

## Results
- Achieved strong predictive performance on race outcomes
- Identified key drivers of race performance (qualifying position, pit strategy, weather)
